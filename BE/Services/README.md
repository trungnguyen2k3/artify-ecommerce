# Services
Thư mục này chứa logic nghiệp vụ (Business Logic) của ứng dụng. Controllers chỉ làm nhiệm vụ tiếp nhận request và trả về response, mọi logic xử lý, tính toán sẽ được viết ở đây.

## Ví dụ cấu trúc:
- `IProductService.cs` (Interface định nghĩa các nghiệp vụ)
- `ProductService.cs` (Lớp thực hiện triển khai interface trên)
